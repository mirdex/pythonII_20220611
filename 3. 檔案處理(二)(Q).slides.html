Excel檔案介紹¶

  • workbook(活頁簿):一個Excel檔案又稱為一個活頁簿檔案
  • worksheet(工作表):每個活頁簿都有多個worksheet可供使用
  • column(欄位):每個worksheet都有A、B...等的欄位名稱
  • row(列):每個worksheet都有1、2...等的列數
  • cell(儲存格):每個worksheet都有儲存格，每個cell以(column, row)表示

巨量欄位Excel¶

  • 在處理巨量資料時，資料的欄位可能大於26欄（A~Z）的英文字母
  • 此時Excel會自動將第27欄命名為AA、第28欄命名為AB...依此類推
  • 為此若我們需要知道欄位名稱對應的索引值或索引值對應的欄位名稱，可使用下列函數
from openpyxl.utils import get_column_letter, column_index_from_string

get_column_letter(index)   #索引轉換欄位名
column_index_from_string('column name')  #欄位名轉換索引

Excel 繪圖¶

  • Excel中常見的Chart有圓餅圖(PieChart)、長條圖(BarChart)、折線圖(LineChart)...等等
  • 每一個圖都需要引入相關方法
    from openpyxl.chart import BarChart, Reference   #以BarChart為例
In [12]:
# 範例

import openpyxl
from openpyxl.chart import BarChart, Reference

excel_data = openpyxl.load_workbook('Files/Score.xlsx')
active_sheet = excel_data["A班"]
chart = BarChart()   # 建構BarChart物件
chart.title = "成績表"
chart.y_axis.title = "分數"
chart.x_axis.title = "程式語言"

data = Reference(active_sheet, min_col = 2, max_col = 3, min_row = 2, max_row = 9)  #B2:C9
chart.add_data(data, titles_from_data=True)
xtitle = Reference(active_sheet, min_col=1, min_row=3, max_row=9)  #建立x軸title
chart.set_categories(xtitle)
active_sheet.add_chart(chart, 'A1')
excel_data.save("Score.xlsx")

JSON資料處理¶

什麼是JSON檔案？¶

  • JSON的全名為Javascript Object Notation
  • JSON大量被應用於Web開發及NOSQL資料庫

JSON資料格式¶

  • 物件(object)
    • 使用大括號{} 將資料包起來
    • 類似於python的字典
    • key必須為字串
    • 例：
      • {"Company":"TSMC", "Class":"SemiConductor"}
  • 陣列(array)
    • 使用中括號[] 將資料包起來
    • 類似於python的串列
    • 其值可以為任意資料型態資料
    • 例：
      • ["Richard", "Man", 25]

import json

json.dumps(data_structor_variable)
